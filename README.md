# Proyecto-Softech
Este proyecto es una base para crear una aplicación que permitira optimizar la gestión interna de una clínica veterinaria, en este caso "Huellitas Felices". Permite registrar, consultar y administrar información sobre pacientes (mascotas), propietarios, citas médicas, tratamientos y personal veterinario.

El objetivo principal del proyecto es mejorar la eficiencia operativa de las clínicas veterinarias, ofreciendo una herramienta intuitiva, moderna y accesible tanto para el personal administrativo como para los veterinarios.
